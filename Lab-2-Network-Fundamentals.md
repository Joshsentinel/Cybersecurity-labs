# Lab 2 - Network Fundamentals

## Objective

Understand local network configuration and internet connectivity.

## Commands Used

ipconfig

ping 8.8.8.8

nslookup google.com

arp -a

tracert 8.8.8.8

## Findings

### Local IP Address

192.168.0.108

### Default Gateway

192.168.0.1

### Subnet Mask

255.255.255.0

### Internet Connectivity

Successful ping responses received from 8.8.8.8.

### DNS Resolution

Successfully resolved google.com.

### ARP Table

Observed MAC address entries for local devices.

### Traceroute

Verified traffic path begins through the default gateway.

## Outcome

Successfully identified local network configuration and verified internet connectivity.
